MongoDB NoSQL 数据库入门 📄

MongoDB 是一个基于文档的 NoSQL 数据库，以其高性能、高可用性和易扩展性著称。本教程将带你全面了解 MongoDB 的核心概念和使用方法。

什么是 MongoDB？

MongoDB 是一个开源的文档数据库，属于 NoSQL 数据库的一种。它以 JSON-like 格式存储数据，支持动态查询和索引。

MongoDB 的特点

  • 文档存储: 使用 BSON 格式存储文档
  • 无模式: 同一集合中的文档可以有不同的字段
  • 高性能: 内存映射文件技术，读写速度快
  • 易扩展: 支持分片，轻松水平扩展
  • 丰富的查询: 支持丰富的查询语言和聚合管道
  • 高可用: 支持副本集，自动故障转移

关系型数据库 vs NoSQL

特性 关系型数据库 MongoDB
数据结构 表格（行、列） 文档（BSON）
模式 固定模式 无模式
扩展方式 垂直扩展 水平扩展
事务 ACID 事务 支持多文档事务
JOIN 支持 支持查找
适用场景 结构化数据 半结构化数据

查询操作符

操作符 描述 示例
$eq 等于 {age: {$eq: 25}}

$lt 小于 {age: {$lt: 30}}

$in 在数组中 {age: {$in: [20, 25, 30]}}
$nin 不在数组中 {age: {$nin: [20, 25]}}
$exists 字段存在 {email: {$exists: true}}
$type 类型匹配 {age: {$type: "int"}}

索引策略

  1. 选择性高的字段: 唯一性高的字段适合建索引
  2. 查询频繁的字段: 经常用于查询条件的字段

  4. 避免过多索引: 索引会影响写入性能
