MySQL 数据库基础入门教程

MySQL 数据库基础入门教程 🗄️

MySQL 是最流行的关系型数据库管理系统之一，本教程将带你从零开始学习 MySQL 的基础知识。

什么是 MySQL？

MySQL 是一个开源的关系型数据库管理系统，由瑞典 MySQL AB 公司开发，现属于 Oracle 公司。它是 Web 应用中最流行的数据库之一。

MySQL 的特点

  • 开源免费: 社区版完全免费
  • 跨平台: 支持 Windows、Linux、macOS
  • 高性能: 查询速度快，支持大数据量
  • 易学易用: SQL 语法简单直观
  • 可靠性高: 支持事务、外键等特性

基本概念

数据库 (Database)

数据库是存储数据的容器，类似于一个文件夹，可以包含多个表。

表 (Table)

表是存储具体数据的结构，类似于 Excel 表格，由行和列组成。

字段 (Field)

字段是表中的一列，代表一种数据类型。

记录 (Record)

记录是表中的一行，代表一条具体的数据。

5. 更新数据 (UPDATE)

6. 删除数据 (DELETE)

数据类型

数值类型

  • INT: 整数
  • BIGINT: 大整数
  • FLOAT: 单精度浮点数
  • DOUBLE: 双精度浮点数
  • DECIMAL: 精确小数

字符串类型

  • CHAR(n): 固定长度字符串
  • VARCHAR(n): 可变长度字符串
  • TEXT: 长文本
  • LONGTEXT: 超长文本

日期时间类型

  • DATE: 日期 (YYYY-MM-DD)
  • TIME: 时间 (HH:MM:SS)
  • DATETIME: 日期时间 (YYYY-MM-DD HH:MM:SS)
  • TIMESTAMP: 时间戳

约束

主键约束 (PRIMARY KEY)

唯一标识表中的每一行记录。

外键约束 (FOREIGN KEY)

建立表与表之间的关联关系。

唯一约束 (UNIQUE)

确保字段值唯一。

非空约束 (NOT NULL)

确保字段不能为空。

高级查询

1. 聚合函数

2. 分组查询 (GROUP BY)

3. 连接查询 (JOIN)

4. 子查询

索引

索引可以大大提高查询速度。

事务

事务用于保证数据的一致性。

性能优化建议

  1. 合理使用索引: 为经常查询的字段创建索引
  2. **避免 SELECT ***: 只查询需要的字段
  3. 使用连接池: 减少数据库连接开销
  4. 定期优化表: OPTIMIZE TABLE table_name
  5. 使用缓存: Redis 等缓存热门数据
  6. 分表分库: 数据量大时考虑分表
